# **App Name**: NFT Showcase

## Core Features:

- NFT Gallery: Display a gallery of NFTs. The NFTs are read from a JSON file.
- NFT Details: Display the details of each NFT, including image, name, description, and owner.
- AI Description: Use AI to generate a short description for each NFT based on its metadata. The model acts as a tool to enhance existing data, if available.
- Filtering and Sorting: Implement basic filtering and sorting options for the NFT gallery.
- User Interface: Create a visually appealing and user-friendly interface for exploring NFTs.

## Style Guidelines:

- Primary color: Dark blue (#1A202C) for a modern feel.
- Secondary color: Light gray (#EDF2F7) for backgrounds and subtle accents.
- Accent: Teal (#4DC0B5) for interactive elements and highlights.
- Clean and minimalist layout with a focus on showcasing NFT images.
- Simple and modern icons for navigation and filtering.
- Subtle transitions and animations for a smooth user experience.
